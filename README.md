# Nashuha & Shafiq Wedding Invitation

A mobile-first Vite and React wedding invitation for Fatin Nashuha and Mohamad Shafiq.

## Run Locally

```powershell
npm install
npm run dev
```

Vite prints the local URL after startup. Create a production bundle with:

```powershell
npm run build
```
